API Overview

The AfriData API provides programmatic access to African datasets, enabling developers to integrate our data repository into their applications, research workflows, and data analysis pipelines.

Base URL

https://afridata.com/api/v1/

Response Format

JSON

Rate Limit

1000 requests/hour

Authentication

AfriData API uses API key authentication. Include your API key in the Authorization header: 

Authorization: Api-Key YOUR_API_KEY_HERE

Available Endpoints

Datasets

GET /api/v1/datasets/

List all datasets with pagination and filtering

Query Parameters: country, category, search, page, page_size
GET /api/v1/datasets/{id}/

Retrieve a specific dataset by ID

POST /api/v1/datasets/

Create a new dataset (authentication required)

GET /api/v1/datasets/popular/

Get popular/trending datasets

Users

GET /api/v1/users/profile/

Get current user profile (authentication required)

POST /api/v1/auth/login/

Authenticate user and get API token

Example Requests

Python (requests)

import requests

# Get all datasets
headers = {'Authorization': 'Api-Key YOUR_API_KEY'}
response = requests.get('https://afridata.com/api/v1/datasets/', headers=headers)
datasets = response.json()

# Filter datasets by country
params = {'country': 'kenya', 'category': 'agriculture'}
response = requests.get('https://afridata.com/api/v1/datasets/', 
                       headers=headers, params=params)

# Create new dataset
data = {
    'name': 'Kenya Population Census 2024',
    'description': 'Latest population data for Kenya',
    'country': 'Kenya',
    'category': 'Demographics'
}
response = requests.post('https://afridata.com/api/v1/datasets/', 
                        headers=headers, json=data)

cURL

# Get datasets filtered by country
curl -H "Authorization: Api-Key YOUR_API_KEY" \
     "https://afridata.com/api/v1/datasets/?country=nigeria"

# Create new dataset
curl -X POST \
     -H "Authorization: Api-Key YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     -d '{"name": "Nigeria GDP Data", "country": "Nigeria"}' \
     https://afridata.com/api/v1/datasets/

JavaScript (fetch)

// Get datasets
fetch('https://afridata.com/api/v1/datasets/', {
    headers: {
        'Authorization': 'Api-Key YOUR_API_KEY'
    }
})
.then(response => response.json())
.then(data => console.log(data));

// Search datasets
fetch('https://afridata.com/api/v1/datasets/?search=agriculture', {
    headers: {
        'Authorization': 'Api-Key YOUR_API_KEY'
    }
})
.then(response => response.json())
.then(data => console.log(data));
